## Getting Started

添加组件：

```
bin/add_module.sh module1
```

中国大陆的优化网速：
```
bin/optimize_cn_network.py android/
bin/optimize_cn_network.py module1/.android/
```

运行组件:

```
bin/run_module.sh module1
```
运行整个app:
```
还是 flutter run
```
